# Cyber Security Fundamentals

## Overview
This repository contains a detailed explanation of core cyber security concepts as part of an academic task.

## Topics Covered
- CIA Triad (Confidentiality, Integrity, Availability)
- Types of attackers
- Attack surfaces
- OWASP Top 10 vulnerabilities
- Mapping daily applications to attack surfaces
- Data flow: User → Application → Server → Database
- Identification of attack points

Through this task, I understood how cyber attacks occur, how applications expose attack surfaces, and how the CIA triad helps in securing systems.
